Expected dataset names

  • Collections: source items, area, costs, batch IDs
  • Extraction: materials recovered by batch and source item
  • Sales: materials sold, buyers, price per kg, payment state
  • Inventory: current stock and value on hand

Beginner-friendly tips

  • Separate files work fine. The uploader will infer the dataset from the filename when possible.
  • Missing columns trigger clear warnings, and valid rows still load where practical.
  • If no inventory file is supplied, the dashboard calculates stock from extraction minus sales.
  • Google Sheets users can use File → Download → CSV or Excel.
